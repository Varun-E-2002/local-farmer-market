# 🌾 Local Farmer Market

A full stack web application that connects farmers directly with buyers.

## 🚀 Live Demo
Frontend: https://your-netlify-url.netlify.app  
Backend API: https://local-farmer-market-1.onrender.com/api/products/

## ✨ Features
- Farmer login and buyer login
- Farmer can upload products
- Buyers can view products
- Add to cart
- Place order
- Track order status
- Role-based authentication
- Live deployed frontend + backend

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Django
- SQLite
- Render
- Netlify
- GitHub
